# Competitor Intelligence (Updated 2026-02-08)

## Google (LearnLM + YouTube + Colab)
- LearnLM: pedagogy-tuned AI model, integrated into Gemini 2.5 (I/O 2025)
- Guided Learning: launched Aug 2025, encourages step-by-step reasoning
- Project Astra: homework tutor prototype (follows along, walks through problems)
- Custom quizzes: interactive quiz generation from any topic
- YouTube integration: LearnLM enhances learning in YouTube already
- Google Skills: AI-driven coding labs with Gemini Code Assist
- $30M committed to AI education
- Free Gemini AI Pro for students through July 2026
- LearnLM outperforms GPT-4o by 31%, Claude 3.5 by 11% on education expert preference

## Microsoft (Copilot + VS Code + Learn)
- GitHub Copilot free tier: 2,000 completions + 50 chat/month
- Copilot Pro free for verified students via GitHub Education
- MS Learn MCP Server (Dec 2025): documentation-in-IDE
- VS Code v1.109 (Jan 2026): multi-agent orchestration hub
- IntelliCode deprecated Dec 2025, all-in on Copilot
- $18/user/month academic offering (Dec 2025)
- Focus: professional developers, not CS students watching lectures

## Replit
- Agent 3 (Sep 2025): explains reasoning, 200 min autonomous work
- Revenue 10x in 9 months post-Agent launch
- $222M raised total
- 30M+ developers, pivoting toward "AI app factory"
- Replit Learn platform launched
- Replit for Teams available for education
- Moving upstream (enterprise), not downstream (university courseware)

## Coursera (Coach)
- Coach: AI tutor powered by Gemini, 1M+ learners
- Socratic dialogue ("Help me practice")
- 9.5% higher quiz pass rate, 11.6% more lessons/hour
- Won Newsweek AI Impact Award 2025
- Available in 26 languages
- Video + AI but no real-time code execution
- Vibe Coding Essentials (by Scrimba) course available on Coursera

## edX (Xpert) / 2U
- Xpert: AI-powered learner tutor, 600K+ users, 350K+ questions asked
- Georgia Tech experimenting with AI-generated courses
- GradyAI: essay grading at scale
- Open edX: AI tutoring via Slack/MS Teams, adaptive learning paths
- 2U filed Chapter 11 in 2024, restructured
- 99M+ registered learners
- CS50 is on edX (key distribution risk for Niotebook)

## Codecademy
- AI Learning Assistant: context-aware, personalized feedback
- AI-powered project hints (review code, personalized hint)
- Code autocomplete for HTML/CSS/JS
- AI interview practice
- Acquired by Skillsoft ~$525M
- 50M+ registered learners
- Self-contained curriculum, no video lectures, no open courseware

## Khan Academy (Khanmigo)
- $4/mo or $44/year for learners
- Free for teachers
- Grew from 68K to 700K users in one year (2023-2025)
- Upgraded from GPT-4 Turbo to GPT-4 Omni
- Supports JS, HTML, Python, SQL coding
- Speech-to-text and text-to-speech
- Multilingual (10+ languages in beta)
- Primary focus: K-12 math/science, not university CS

## freeCodeCamp
- Nonprofit, donation-funded
- 40K+ job placements
- 65-hour GenAI bootcamp (YouTube)
- Interactive web exercises (text-based)
- No AI tutor feature on platform
- No video-integrated coding

## Brilliant.org
- ~$90M raised, 10M+ learners
- Interactive problem-solving (gamified)
- Human-crafted content ("not AI")
- Expanding CS through 2026 (algorithms, data structures)
- No video, no AI tutor, no open courseware
- Fundamentally different pedagogical model

## Scrimba (CRITICAL COMPETITOR)
- YC S20, ~$940K raised
- 120K MAU, 500K+ students served
- $1.9M revenue 2024
- "Scrim" format: video + inline code editor (pause, edit, run)
- AI feedback on code
- Bootcamp-style courses (frontend, AI)
- Creates own content (not open courseware)
- 4.9/5 rating
- Most structurally similar to Niotebook
