# Niotebook v0.2

AI-native programming learning interface that synchronizes Video + Code + AI.

## Getting Started

- Ensure Bun 1.1.x is installed and `~/.bun/bin` is on your PATH (run `exec /bin/zsh` after install).
- Install dependencies: `bun install`
- Start dev server: `bun run dev`
- Open `http://localhost:3000`
